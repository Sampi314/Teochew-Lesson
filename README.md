# Học Tiếng Triều Châu - Teochew Learning

Ứng dụng web học tiếng Triều Châu (Teochew) dành cho người Việt Nam.

## Tính năng

- **Từ vựng theo chủ đề**: Chào hỏi, số đếm, gia đình, thức ăn, giao tiếp hàng ngày, và nhiều hơn nữa
- **Hệ thống thanh điệu**: So sánh 8 thanh Triều Châu với 6 thanh tiếng Việt
- **Bài kiểm tra**: Trắc nghiệm tương tác với nhiều dạng câu hỏi
- **Từ Hán-Việt**: Mỗi từ đều kèm theo từ Hán-Việt liên quan giúp dễ nhớ
- **Từ vựng hôm nay**: Mỗi ngày một từ mới trên trang chủ

## Sử dụng

Truy cập ứng dụng tại: `https://<username>.github.io/Teochew-Lesson/`

## Cài đặt GitHub Pages

1. Vào **Settings** của repository
2. Chọn **Pages** ở menu bên trái
3. Trong phần **Source**, chọn branch `main` và thư mục `/ (root)`
4. Nhấn **Save**
